# Data Cleaning & Reporting Automation

## Project Overview
This project automates data cleaning and reporting workflows using Python. It handles missing values, removes duplicate records, cleans inconsistent data, and generates summary reports.

## Features
- Remove duplicate records
- Handle missing values
- Clean inconsistent data
- Generate automated summary reports
- Export cleaned dataset

## Technologies Used
- Python
- Pandas
- CSV

## Files
- data_cleaning.py
- sales_data.csv
- cleaned_sales_data.csv
- report.txt

## Outcome
Improves data quality and reduces manual effort in reporting and analysis.
